# lafosse-joseph-angelique

(Binôme 85 : Léon Lafosse, Joran Joseph-Angélique)

Ce projet, effectué dans le cadre du cours de Langage Objet Avancé, correspond à une démo du puzzle-game Baba is You (2019, Arvi Teikari); il présente quelques règles essentielles du jeu original et des assets de celui-ci ont été utilisés.

Instructions :
- Décompresser le fichier
- make all
- exécuter avec ./baba_is_you
- jouer (souris et flèches directionnelles).
